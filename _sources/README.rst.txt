**Calculator** is written using Python and the
`PyQt5 <https://www.riverbankcomputing.com/software/pyqt/>`__ framework.

Installation (Python 3.6+)
--------------------------

The ``calculator`` package is available from
`PyPI <https://pypi.org/project/Calculator/>`__ and requires Python
3.6+. It offers a simple `graphical-user
interface <https://en.wikipedia.org/wiki/Graphical_user_interface>`__
(GUI).

We recommend using `pipx <https://github.com/pipxproject/pipx>`__.
Follow the `pipx
installation <https://pipxproject.github.io/pipx/installation/>`__
guide, then proceed to install the ``calculator`` package with ``pipx``,

.. code:: shell

   pipx install calculator

The ``calculator`` `command-line
interface <https://en.wikipedia.org/wiki/Command-line_interface>`__
(CLI) utility is now globally available to start the builtin GUI, but
remains isolated and easily manageable via the excellent ``pipx`` CLI.
